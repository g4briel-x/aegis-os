# 1. Purpose

Aegis OS is a structured operating system for AI-assisted product, engineering, security, infrastructure, operations and business execution.

It organizes reusable knowledge into:

```text
docs
skills
playbooks
patterns
templates
agents
scripts
```

The current foundation focuses on documentation, skills and playbooks.
