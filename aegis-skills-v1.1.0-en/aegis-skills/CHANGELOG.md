# Changelog

All notable changes to this project are documented in this file.

## [1.1.0] - 2026-07-20

### Added

- Complete English edition of all 37 skills and repository documentation.
- English trigger examples, evaluation expectations, response templates, quality gates, and operational controls.
- Translation and language-validation reports.

### Changed

- Updated the library, tooling, manifest, registry, and distribution versions to `1.1.0`.
- Rebuilt every standalone Claude skill ZIP from the English sources.
- Preserved stable skill identifiers, directory names, file names, paths, schema keys, and Python APIs.

## [1.0.0] - 2026-07-20

### Added

- Migrated 36 historical role profiles into standalone skills and added the Aegis Software Studio orchestrator.
- Added valid frontmatter, manifests, registries, schemas, evaluations, tests, reports, and reproducible ZIP packaging.
- Replaced ad hoc platform automation with a testable Python CLI.
- Added CI validation and eliminated embedded `.git` directories and PowerShell scripts from distributions.
