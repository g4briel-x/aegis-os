# Translation Report

## Scope

This report covers the conversion of Aegis Skills from the French content edition to the English content edition `1.1.0`.

## Translated content

- 37 `SKILL.md` entry points.
- 37 manifests and registry descriptions.
- 37 professional profiles.
- 37 detailed workflows.
- 37 quality-gate references.
- 37 operational-control references.
- 37 bibliography introductions.
- 37 prompt-example files.
- 37 response templates.
- 37 evaluation files.
- Repository README, contribution guide, changelog, standards, templates, categories, and reports.

## Preserved technical contracts

The following elements were intentionally not renamed:

- skill identifiers;
- directory and file names;
- JSON and YAML keys;
- Python modules, classes, and functions;
- commands and CLI options;
- internal paths and URLs;
- schema identifiers;
- historical source paths recorded for traceability.

## Quality approach

The conversion uses role-specific English missions and deliverables, normalized professional terminology, consistent output contracts, and regenerated evaluation prompts. It is not a word-for-word translation; wording was rewritten to remain precise, idiomatic, and operationally useful.
