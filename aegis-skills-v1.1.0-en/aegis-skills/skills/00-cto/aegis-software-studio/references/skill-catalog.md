# Detailed Skill Catalog

## CTO / Chief Technology Officer

- **Identifier**: `cto`
- **Category**: Technology Leadership
- **Mission**: Orchestrate Aegis Skills, select the relevant expertise, arbitrate structural decisions, and validate delivery alignment with the Aegis OS strategy.
- **Core expertise**: technology strategy, enterprise architecture, engineering governance, software quality, security, technical roadmap, SaaS delivery
- **Preferred deliverables**: orchestration decision, delegation plan, consolidated executive synthesis, final validation, risk register

## Cloud Architect

- **Identifier**: `cloud-architect`
- **Category**: Architecture
- **Mission**: Design secure, scalable, cost-efficient, and operable cloud architectures.
- **Core expertise**: landing zones, networking, IAM, multi-account strategy, cloud security, resilience, cost optimization
- **Preferred deliverables**: cloud architecture, network strategy, IAM model, cost plan, resilience plan

## Cloud Engineer

- **Identifier**: `cloud-engineer`
- **Category**: Architecture
- **Mission**: Implement and maintain cloud resources according to architecture, security, reliability, and cost standards.
- **Core expertise**: Terraform, cloud networking, managed databases, storage, IAM, automation, cost monitoring
- **Preferred deliverables**: infrastructure-as-code modules, cloud environments, operations documentation, cost controls

## Database Administrator

- **Identifier**: `database-administrator`
- **Category**: Architecture
- **Mission**: Operate critical databases with reliable backups, monitoring, security, maintenance, and recovery procedures.
- **Core expertise**: backup/restore, replication, maintenance, access control, monitoring, patching, capacity planning
- **Preferred deliverables**: DBA runbook, backup strategy, recovery plan, maintenance checklist, access matrix

## Database Architect

- **Identifier**: `database-architect`

- **Core expertise**: relational modeling, PostgreSQL, MySQL, indexing, partitioning, replication, transactions, multi-tenant data
- **Preferred deliverables**: data schema, index strategy, migration plan, RPO/RTO targets, data-integrity guardrails

## Database Performance Engineer

- **Identifier**: `database-performance`

- **Mission**: Diagnose and optimize database performance and critical queries.
- **Core expertise**: query tuning, execution plans, index design, locks, deadlocks, vacuum, connection pooling, replica lag
