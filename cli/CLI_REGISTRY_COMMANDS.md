# Registry commands

```console
aegis --repo-root . registry list
aegis --repo-root . registry domains
aegis --repo-root . registry tags
aegis --repo-root . docs list
aegis --repo-root . release status
```

Generic asset filters replace separate commands for every asset family:

```console
aegis --repo-root . asset type skill
aegis --repo-root . asset type playbook
aegis --repo-root . asset type pattern
aegis --repo-root . asset type template
```
