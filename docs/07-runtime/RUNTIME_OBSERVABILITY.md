1. Introduction

Runtime Observability defines the live signals collected from runtime operations.

It allows the system to understand how execution behaves in real conditions.

2. Runtime Signals

Observe:

execution logs;
task durations;
queue lengths;
retries;
failures;
state transitions;
resource usage.
