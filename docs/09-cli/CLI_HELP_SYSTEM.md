Aegis OS — CLI Help System

Version: 0.1
Status: CLI Guide

1. Introduction

This document defines how help and discoverability should work in the CLI.

A strong help system reduces onboarding friction and operational mistakes.

2. Help Principles

Help should be:

contextual;
concise;
progressive;
example-driven.
3. Example Commands
aegis help
aegis skill --help
aegis workflow run --help
aegis doctor --help
4. Help Contents

Each help screen should show:

command purpose;
usage syntax;
options;
examples;
exit codes when relevant.
5. Final Principle

Help is part of the interface, not an afterthought.
