# Aegis OS — Deprecation Authority Model

Version: 0.1.0  
Status: Draft

## Purpose

This document defines how assets are deprecated or retired in Aegis OS.

## Deprecation Reasons

An asset may be deprecated when it is:

```text
outdated
duplicated
unsafe
replaced by a better asset
misaligned with architecture
too incomplete for continued use
```

## Deprecation Requirements

A deprecation decision should include:

```text
asset id
reason
replacement if available
effective version
migration notes
owner approval
registry update
```

## Deprecation States

```text
active
deprecated
retired
removed
```

## Final Principle

> Deprecation should be deliberate, traceable and helpful to users.
