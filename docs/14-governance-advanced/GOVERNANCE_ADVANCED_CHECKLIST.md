# Aegis OS — Advanced Governance Checklist

Version: 0.1.0  
Status: Draft

## Purpose

This checklist helps verify whether advanced governance is in place.

## Checklist

```text
[ ] Ownership model exists
[ ] Approval chain model exists
[ ] Trust level model exists
[ ] Certification model exists
[ ] Maturity model exists
[ ] Domain governance model exists
[ ] Policy enforcement model exists
[ ] Deprecation authority model exists
[ ] Governance audit model exists
[ ] Ecosystem steering model exists
[ ] Registry references this folder
[ ] Validation paths pass
```

## Release Use

Before closing a major release, confirm:

```text
[ ] Governance changes are documented
[ ] Release decision is traceable
[ ] Deprecated assets are marked
[ ] New assets have owners
[ ] Validation passes
```

## Final Principle

> Governance is complete only when it can be checked.
