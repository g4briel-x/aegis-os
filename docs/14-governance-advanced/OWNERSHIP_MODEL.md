# Aegis OS — Ownership Model

Version: 0.1.0  
Status: Draft

## Purpose

This document defines how ownership works across Aegis OS assets.

## Owned Assets

Ownership applies to:

```text
docs
skills
playbooks
patterns
templates
registries
scripts
CLI commands
workflows
reports
```

## Owner Responsibilities

An owner is responsible for:

```text
asset accuracy
maintenance
review
registry metadata
compatibility
deprecation decisions
documentation quality
```

## Ownership Types

```text
individual owner
domain owner
technical owner
governance owner
release owner
```

## Ownership Rule

Every important asset should have a clear owner or owning domain.

## Final Principle

> Assets without ownership become stale, inconsistent and risky.
