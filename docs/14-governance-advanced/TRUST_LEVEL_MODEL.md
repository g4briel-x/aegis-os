# Aegis OS — Trust Level Model

Version: 0.1.0  
Status: Draft

## Purpose

This document defines trust levels for Aegis OS assets.

## Trust Levels

```text
experimental
draft
usable
reviewed
certified
deprecated
retired
```

## Meaning

Experimental:

```text
Idea or prototype. Not ready for operational use.
```

Draft:

```text
Structured but still incomplete or under review.
```

Usable:

```text
Can be used locally with reasonable confidence.
```

Reviewed:

```text
Reviewed by the appropriate owner or domain.
```

Certified:

```text
Approved for official use in stable workflows.
```

Deprecated:

```text
Still available but should not be used for new work.
```

Retired:

```text
Removed from active use.
```

## Final Principle

> Trust levels help users know which assets are safe to depend on.
