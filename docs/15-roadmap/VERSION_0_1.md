# Aegis OS — Version 0.1 Roadmap

Version: 0.1  
Status: Version Roadmap

---

# 1. Objective

Aegis OS v0.1 establishes the documentation foundation.

---

# 2. Scope

v0.1 includes:

- foundation documents;
- architecture documents;
- specifications;
- governance;
- reference models;
- development workflows;
- runtime models;
- SDK, CLI, Marketplace, AI, operations and security architecture docs.

---

# 3. Success Criteria

v0.1 succeeds when:

- documentation structure is coherent;
- directory model is stable;
- lifecycle, runtime and governance are defined;
- future implementation work has a clear specification base.

---

# 4. Out of Scope

v0.1 does not require:

- full runtime implementation;
- production-grade CLI;
- full SDK implementation;
- Marketplace deployment.

---

# 5. Final Principle

> v0.1 is the foundation release: architecture first, implementation later.
