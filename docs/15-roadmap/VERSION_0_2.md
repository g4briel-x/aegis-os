# Aegis OS — Version 0.2 Roadmap

Version: 0.1  
Status: Version Roadmap

---

# 1. Objective

Aegis OS v0.2 establishes the first usable framework layer.

---

# 2. Scope

v0.2 should include:

- stabilized `core/`;
- stabilized `shared/`;
- Skill v2 structure;
- first premium Skills;
- first reusable Playbooks;
- essential Templates;
- registry conventions.

---

# 3. Priority Deliverables

Examples:

- 5 to 10 premium Skills;
- 5 to 10 Playbooks;
- core orchestration definitions;
- shared engineering knowledge base;
- first examples directory.

---

# 4. Success Criteria

v0.2 succeeds when:

- Aegis OS can be used as a structured expert framework;
- Skills are no longer placeholders;
- documentation and component structure align;
- a contributor can build new Skills using stable guidance.

---

# 5. Final Principle
> v0.2 is the first framework-usable release.
