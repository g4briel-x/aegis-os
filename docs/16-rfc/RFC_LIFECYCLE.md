# Aegis OS — RFC Lifecycle

Version: 0.1  
Status: RFC Document

---

# 1. Introduction

This document defines the lifecycle of an RFC in Aegis OS.

---

# 2. Lifecycle Stages

```text
Idea

 ↓

Draft RFC

 ↓

Review

 ↓

Revision

 ↓

Approval or Rejection

 ↓

Accepted RFC

 ↓

Implementation Tracking

 ↓

Completed / Superseded / Withdrawn
```

---

# 3. Lifecycle States

Suggested states:

- draft;
- under-review;
- revision-required;
- accepted;
- rejected;
- withdrawn;
- superseded;
- implemented.

---

# 4. Final Principle

> An RFC should remain traceable from proposal to final outcome.
