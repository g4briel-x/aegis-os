# Aegis OS — Getting Started Guide

Version: 0.1  
Status: Adoption Guide

---

# 1. Introduction

This guide helps new users understand and start using Aegis OS.

---

# 2. First Steps

Recommended onboarding path:

1. Read the project context.
2. Review the system architecture.
3. Understand Skills and Playbooks.
4. Explore the documentation structure.
5. Try a simple use case.
6. Create or adapt a first Skill.

# 4. First Practical Task

Suggested first task:

- choose one professional role;
- create a Skill proposal;
- validate it against the Skill specification;
- add examples;
- review quality gates.
