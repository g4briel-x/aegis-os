# Aegis OS — Playbook Review Checklist

Version: 0.1  
Status: Playbook Checklist

---

# 1. Playbook Review Checklist

```text
[ ] Playbook has clear identity
[ ] Metadata is valid
[ ] Trigger is clearly defined
[ ] Scope and non-scope are documented
[ ] Required inputs are listed
[ ] Expected outputs are defined
[ ] Steps are executable
[ ] Decision points are useful
[ ] Validation checks are present
[ ] Completion criteria are explicit
[ ] Examples are realistic
[ ] Related Skills are referenced
```

---

# 2. Final Principle

> Review protects the Playbook ecosystem from procedures that look useful but cannot be executed reliably.
