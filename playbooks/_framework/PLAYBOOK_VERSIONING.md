# Aegis OS — Playbook Versioning

Version: 0.1  
Status: Playbook Standard

---

# 1. Introduction

This document defines versioning rules for Playbooks.

---

# 2. Version Format

Recommended format:

```text
MAJOR.MINOR.PATCH
```

Example:

```text
0.1.0
```

---

# 3. Version Rules

Increment:

- PATCH for small wording or checklist corrections;
- MINOR for new compatible steps, examples or outputs;
- MAJOR for breaking workflow or output changes.

---

# 4. Final Principle

> Playbook versioning protects users from silent changes in execution behavior.
