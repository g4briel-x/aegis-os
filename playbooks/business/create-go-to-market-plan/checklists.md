# Create Go-To-Market Plan — Checklists

Version: 0.1.0  
Status: Premium Draft

---

# 1. Segment Checklist

```text
[ ] Target segment defined
[ ] Buyer identified
[ ] User identified
[ ] Pain point defined
[ ] Budget assumption documented
[ ] Reachable channel identified
[ ] First segment prioritized
```

---

# 2. Positioning Checklist

```text
[ ] Product category defined
[ ] Problem statement clear
[ ] Outcome clear
[ ] Differentiator clear
[ ] Proof or credibility identified
[ ] Objections anticipated
[ ] Message tested or ready to test
```

---

# 3. Offer Checklist

```text
[ ] Offer type selected
[ ] Call to action defined
[ ] Pricing or access model clear
[ ] Expected customer commitment defined
[ ] Trial or pilot terms documented
[ ] Next step after CTA defined
[ ] Follow-up path ready
```

---

# 4. Channel Checklist

```text
[ ] Primary channel selected
[ ] Secondary channel selected if needed
[ ] Audience source identified
[ ] Asset needs identified
[ ] Outreach list or traffic source prepared
[ ] Campaign owner assigned
[ ] Test budget or effort defined
```

---

# 5. Funnel Metrics Checklist

```text
[ ] Reach metric defined
[ ] Conversion metric defined
[ ] Activation metric defined
[ ] Revenue or pilot metric defined
[ ] Review cadence defined
[ ] Tracking method selected
[ ] Success threshold defined
```

---

# 6. Execution Checklist

```text
[ ] Weekly actions defined
[ ] Owners assigned
[ ] Launch assets ready
[ ] Outreach cadence ready
[ ] Feedback capture ready
[ ] Decision checkpoint scheduled
[ ] Risks documented
```

---

# 7. Final Principle

> GTM checklists make execution practical, measurable and reviewable.
