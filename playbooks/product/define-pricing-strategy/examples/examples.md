# 1. Example — Audiovisual Financing SaaS

## Trigger

A platform helps creators move from audiovisual project idea to financing.

## Expected Execution

The Playbook should guide the team to:

- segment creators, studios, producers and investors;
- identify value drivers such as faster financing readiness and investor visibility;
- compare per-project, per-seat and subscription models;
- create Starter, Pro and Studio tiers;
- define paid pilot or demo-led validation.

## Expected Output

```text
Customer segment map
Value driver map
Pricing metric recommendation
Tier structure
Pricing experiment plan
Launch pricing recommendation
```

# 2. Example — B2B Workflow Automation SaaS

## Trigger

A workflow SaaS needs monetization before MVP launch.

## Expected Execution

The Playbook should guide the team to:

- define team size segments;
- evaluate per-seat pricing;
- package automation limits by tier;
- create validation calls with target buyers.

---

# 3. Example — AI-Powered SaaS Tool

## Trigger

A SaaS product uses AI API calls with variable costs.

## Expected Execution

The Playbook should guide the team to:

- review AI cost per usage;
- define usage limits;
- prevent margin loss;
- design add-on credits or usage-based overage.

---

# 4. Example — Enterprise SaaS Pilot

## Trigger

An enterprise customer wants a custom pilot.

## Expected Execution

The Playbook should guide the team to:

- define paid pilot scope;
- set success criteria;
- define conversion pricing;
- document custom support and onboarding cost.

---

# 5. Final Principle

> Examples show that SaaS pricing must connect segment, value, package, metric and validation.
