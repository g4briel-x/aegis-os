# Prepare SaaS Launch — Checklists

Version: 0.1.0  
Status: Premium Draft

---

# 1. Launch Strategy Checklist

```text
[ ] Launch type defined
[ ] Launch goal defined
[ ] Target audience defined
[ ] Launch owner assigned
[ ] Target date selected
[ ] Success metrics defined
[ ] Constraints documented
```

---

# 2. Product Readiness Checklist

```text
[ ] MVP scope confirmed
[ ] Critical workflows tested
[ ] Known blockers resolved
[ ] Release plan ready
[ ] Production monitoring ready
[ ] Rollback path defined
[ ] Security risks reviewed
```

---

# 3. Offer and Pricing Checklist

```text
[ ] Value proposition clear
[ ] Pricing strategy ready
[ ] Packaging clear
[ ] Trial or pilot model defined
[ ] Billing flow ready if needed
[ ] FAQ prepared
[ ] Upgrade path defined
```

---

# 4. Onboarding Checklist

```text
[ ] Signup works
[ ] Invitation flow works if needed
[ ] First-run path defined
[ ] Empty states reviewed
[ ] Help content prepared
[ ] Activation milestone defined
[ ] Support prompt available
```

---

# 5. Launch Asset Checklist

```text
[ ] Landing page ready
[ ] Announcement copy ready
[ ] Email copy ready if needed
[ ] Demo script ready
[ ] Screenshots or video ready
[ ] FAQ ready
[ ] Terms or policy links ready if needed
```

---

# 6. Support and Feedback Checklist

```text
[ ] Support channel active
[ ] Feedback channel active
[ ] Known issues documented
[ ] Escalation owner assigned
[ ] User interview plan ready
[ ] Bug reporting path ready
[ ] Response expectations defined
```

---

# 7. Analytics Checklist

```text
[ ] Signup tracked
[ ] Activation tracked
[ ] Conversion tracked if needed
[ ] Feature usage tracked
[ ] Error monitoring active
[ ] Support volume tracked
[ ] Review cadence defined
```

# 8. Final Principle

> Launch checklists protect the team from confusing shipping with market readiness.
