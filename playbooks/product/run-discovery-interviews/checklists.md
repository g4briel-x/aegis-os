# Run Discovery Interviews — Checklists

Version: 0.1.0  
Status: Premium Draft

---

# 1. Preparation Checklist

```text
[ ] Discovery objective defined
[ ] Target customer profile defined
[ ] Assumptions listed
[ ] Interview guide prepared
[ ] Participant list created
[ ] Note-taking format prepared
[ ] Consent or recording approach considered
```

---

# 2. Interview Quality Checklist

```text
[ ] Questions focus on real past behavior
[ ] Leading questions avoided
[ ] Participant describes current workflow
[ ] Pain intensity explored
[ ] Current workaround identified
[ ] Decision process explored
[ ] Buying signal explored
[ ] Objections captured
```

---

# 3. Evidence Checklist

```text
[ ] Quotes or close paraphrases captured
[ ] Pain described
[ ] Frequency captured
[ ] Severity captured
[ ] Current alternative captured
[ ] Cost of problem captured where possible
[ ] Follow-up interest captured
```

---

# 4. Synthesis Checklist

```text
[ ] Patterns grouped
[ ] Assumptions validated or invalidated
[ ] Strongest pains identified
[ ] Weak signals separated from strong signals
[ ] Segment differences noted
[ ] Product decision recommended
[ ] Next step defined
```

---

# 5. Final Principle

> Interview checklists prevent discovery from becoming casual conversation with no decision value.
