# Aegis OS — Skills Framework v2

Version: 0.2  
Status: Framework Standard

---

# 1. Purpose

The Skills Framework v2 defines how expert capabilities are created, structured, reviewed and maintained inside Aegis OS.

A Skill is not a simple prompt. A Skill is a reusable expert capability with identity, scope, workflows, checklists, prompts, examples and quality gates.

---

# 2. Core Principle

> A Skill must behave like a professional expert system, not like a loose collection of instructions.

---

# 3. Framework Scope

This framework defines:

- Skill folder structure;
- Skill metadata;
- authoring standards;
- quality gates;
- review process;
- versioning rules;
- certification readiness;
- reusable templates.

---

# 4. Standard Skill Files

A complete Skill v2 should contain:

```text
README.md
SKILL.md
metadata.yaml
expertise.md
workflows.md
checklists.md
prompts.md
examples/examples.md
```

---

# 5. Final Principle

> Skills v2 makes expert behavior structured, reusable and auditable.
