# Aegis OS — Skill Examples Guide

Version: 0.2  
Status: Skill Guide

---

# 1. Introduction

This guide defines how examples should be written for Skills.

Examples show how the Skill behaves in realistic scenarios.

---

# 2. Example Requirements

Examples should include:

- user request;
- context;
- expected Skill behavior;
- output structure;
- validation notes.

---

# 3. Example Types

Useful examples:

- simple task;
- complex task;
- troubleshooting task;
- review task;
- edge case.

---

# 4. Final Principle

> Examples make a Skill easier to understand, test and reuse.
