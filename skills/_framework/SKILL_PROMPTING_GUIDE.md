# Aegis OS — Skill Prompting Guide

Version: 0.2  
Status: Skill Guide

---

# 1. Introduction

This guide defines how prompts should be written inside Skills v2.

---

# 2. Prompting Philosophy

Aegis OS follows this principle:

> Skill prompts should activate structured expert behavior, not produce generic responses.

---

# 3. Prompt Types

Recommended prompt types:

- diagnostic prompt;
- planning prompt;
- execution prompt;
- review prompt;
- correction prompt;
- final delivery prompt.

---

# 4. Prompt Structure

A reusable prompt should define:

```text
Role
Context
Task
Constraints
Output format
Validation requirements
```

---

# 5. Final Principle

> A strong Skill prompt makes expert behavior repeatable.
