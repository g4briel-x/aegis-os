# Aegis OS — Skill v2 Specification

Version: 0.2  
Status: Skill Standard

---

# 1. Introduction

This specification defines the required structure and quality expectations for Aegis OS Skills v2.

---

# 2. Definition

A Skill is a packaged expert capability designed to help an AI system perform a professional role or task with structured reasoning, domain knowledge and validation.

---

# 3. Skill Requirements

A Skill v2 must define:

- identity;
- role;
- domain;
- scope;
- inputs;
- outputs;
- workflows;
- decision logic;
- quality standards;
- examples.
