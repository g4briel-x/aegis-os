# Aegis OS — Skill Versioning

Version: 0.2  
Status: Skill Standard

---

# 1. Introduction

This document defines versioning rules for Skills v2.

---

# 2. Version Format

Recommended format:

```text
MAJOR.MINOR.PATCH
```

Example:

```text
0.2.0
```

---

# 3. Version Rules

Increment:

- PATCH for small corrections;
- MINOR for new compatible capability;
- MAJOR for breaking structural or behavioral changes.

---

# 4. Final Principle

> Skill versioning protects users from silent behavior changes.
