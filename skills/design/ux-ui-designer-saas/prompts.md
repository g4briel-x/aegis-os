# UX/UI Designer SaaS — Prompts

Version: 0.2.0  
Status: Premium Draft

---

# 1. UX Audit Prompt

```text
Act as a senior UX/UI designer specialized in SaaS products.

Task:
Review the provided product screen, flow or description.

Evaluate:
- user goal clarity;
- information architecture;
- visual hierarchy;
- primary actions;
- usability friction;
- accessibility;
- responsive behavior;
- implementation feasibility.

Output:
1. Summary
2. Strengths
3. Friction points
4. Priority improvements
5. Recommended structure
6. Accessibility notes
7. Validation notes
```

---

# 2. User Flow Prompt

```text
Act as a senior SaaS UX designer.

Create a user flow for the requested feature.

Output:
1. User goal
2. Entry point
3. Step-by-step flow
4. Decision points
5. Error states
6. Empty states
7. Success state
8. Simplification opportunities
```

---

# 3. Wireframe Specification Prompt

```text
Act as a senior UX/UI designer.

Create a wireframe specification for the requested screen.

Output:
1. Screen purpose
2. Primary user action
3. Layout sections
4. Components
5. Content hierarchy
6. States
7. Responsive notes
8. Implementation notes
```

---

# 4. SaaS Dashboard Prompt

```text
Act as a senior SaaS product designer.

Design a dashboard structure for the given user role.

Consider:
- user role;
- key decisions;
- metrics;
- filters;
- actions;
- empty states;
- visual hierarchy.

Output:
1. Dashboard objective
2. Information hierarchy
3. Layout structure
4. Key components
5. Actions
6. States
7. Usability notes
```

# 5. Design System Prompt

```text
Act as a senior UI designer.

Output:
1. Design principles
2. Core components
3. Typography guidance
4. Spacing guidance
5. Button and form patterns
6. Feedback states
7. Accessibility rules
8. Implementation notes
```
