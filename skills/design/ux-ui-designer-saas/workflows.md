# UX/UI Designer SaaS — Workflows

Version: 0.2.0  
Status: Premium Draft

---

# 1. UX Discovery Workflow

Use this workflow when starting from a product idea or feature.

```text
1. Identify target users
2. Define user goals
3. Define business goals
4. Identify main use cases
5. Map friction points
6. Define success moments
7. Recommend UX direction
```

---

# 2. User Flow Workflow

Use this workflow when designing a journey.

```text
1. Define entry point
2. Define user intent
3. Map required steps
4. Identify decisions and branches
5. Define success state
6. Define error and empty states
7. Simplify unnecessary steps
8. Document final flow
```

---

# 3. Wireframe Workflow

Use this workflow when designing a screen or interface.

```text
1. Define screen purpose
2. Identify primary user action
3. Define information hierarchy
4. Place key content zones
5. Define components
6. Define states
7. Review responsiveness
8. Validate usability
```

---

# 4. Dashboard Design Workflow

Use this workflow for SaaS dashboards.

```text
1. Define dashboard user role
2. Define decisions the dashboard supports
3. Select key metrics or objects
4. Group information by priority
5. Define filters and actions
6. Define empty and loading states
7. Review visual hierarchy
8. Validate actionability
```

---

# 5. UX Audit Workflow

Use this workflow when reviewing an existing interface.

```text
1. Identify user goal
2. Review flow clarity
3. Review navigation
4. Review visual hierarchy
5. Review form and action clarity
6. Review accessibility
7. Identify friction
8. Recommend prioritized improvements
```

---

# 6. 4-Pass UX/UI Validation Workflow

Every major UX/UI output must be reviewed using this workflow:

```text
Pass 1 — User goal and journey alignment
Pass 2 — Information architecture and usability review
Pass 3 — Visual hierarchy, accessibility and responsiveness review
Pass 4 — Product clarity, conversion and implementation readiness review
```

---

# 7. Final Principle

> UX/UI work is complete only when the design supports the user's next best action clearly.
