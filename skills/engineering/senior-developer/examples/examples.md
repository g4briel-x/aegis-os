# 1. Example — Python Script

## User Request

Create a Python script that reads a CSV file and outputs the number of rows per category.

## Expected Skill Behavior

The Skill should:

- ask for the column name if missing;
- provide readable Python code;
- handle missing file or missing column errors;
- explain how to run the script;
- perform 4-pass verification.

## Expected Output Structure

```text
Assumptions
Implementation
Code
Verification
Usage
```

# 2. Example — SQL Query

## User Request

Write a SQL query to list customers who placed more than 3 orders in the last 30 days.

## Expected Skill Behavior

The Skill should:

- assume table names only if not provided;
- state assumptions;
- write clear SQL;
- consider date filtering;
- explain indexes that may help.

# 5. Example — Low-Code Integration

## User Request

I use Airtable as a database and Glide as a frontend. How should I structure my data for a client tracking app?

## Expected Skill Behavior

The Skill should:

- identify core tables;
- define relationships;
- warn about Airtable limitations;
- explain how Glide will consume the data;
- propose a simple scalable structure.
