# 1. Example — GitHub Actions Pipeline

## User Request

Create a GitHub Actions pipeline for a Node.js application.

## Expected Skill Behavior

The Skill should:

- define install step;
- define test step;
- define build step;
- avoid exposing secrets;
- optionally define deployment stage;
- include validation notes.

## Expected Output Structure

```text
Assumptions
Pipeline overview
Workflow YAML
Secrets and environment notes
Validation notes
```
