# Technical Project Manager — Workflows

Version: 0.2.0  
Status: Premium Draft

---

# 1. Project Planning Workflow

Use this workflow when starting or restructuring a technical project.

```text
1. Clarify project objective
2. Define scope and non-scope
3. Identify stakeholders
4. Identify deliverables
5. Break work into milestones
6. Identify dependencies
7. Identify risks
8. Define communication and tracking model
```

---

# 2. Milestone Planning Workflow

Use this workflow when defining delivery milestones.

```text
1. Define target outcome
2. Identify required deliverables
3. Sequence work by dependency
4. Assign owners
5. Define completion criteria
6. Identify risks and blockers
7. Define review points
8. Confirm milestone readiness
```

---

# 3. Status Reporting Workflow

Use this workflow when creating a project update.

```text
1. Summarize current status
2. Highlight completed work
3. Identify active work
4. Identify blockers
5. Identify risks
6. List decisions needed
7. List next actions
8. Tailor message to audience
```

---

# 4. Risk Management Workflow

Use this workflow when reviewing delivery risk.

```text
1. Identify risk
2. Define impact
3. Define likelihood
4. Identify owner
5. Define mitigation
6. Define escalation threshold
7. Track status
8. Review regularly
```

---

# 5. Release Coordination Workflow

Use this workflow when preparing a release.

```text
1. Confirm release scope
2. Confirm acceptance criteria
3. Confirm QA status
4. Confirm deployment readiness
5. Confirm rollback plan
6. Confirm communication plan
7. Monitor release
8. Capture post-release actions
```

---

# 6. 4-Pass Project Validation Workflow

Every major project management output must be reviewed using this workflow:

```text
Pass 1 — Scope, objective and stakeholder alignment
Pass 2 — Timeline, dependency and resource review
Pass 3 — Risk, blocker and decision review
Pass 4 — Communication, ownership and execution-readiness review
```

---

# 7. Final Principle

> Project workflow is complete only when ownership, next actions and risks are clear.
