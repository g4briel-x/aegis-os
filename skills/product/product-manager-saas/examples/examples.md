# 1. Example — Audiovisual SaaS

## User Request

Create a SaaS product strategy for a platform that helps film, series and documentary projects move from idea to financing.

## Expected Skill Behavior

The Skill should:

- identify creator, producer and investor personas;
- define core pain points;
- propose MVP modules;
- define monetization options;
- define success metrics;
- sequence the roadmap;
- avoid overbuilding the first version.

## Expected Output Structure

```text
Assumptions
Target users
Problem
Value proposition
MVP scope
Business model
Roadmap
Metrics
Risks
```

# 2. Example — MVP Scope

## User Request

I want to launch a SaaS for freelancers to manage clients, invoices and project tasks. What should the MVP include?

## Expected Skill Behavior

The Skill should:

- identify the primary user;
- focus on core workflow;
- select must-have features;
- exclude advanced automation;
- define launch metrics.

# 3. Example — Feature Prioritization

## User Request

Prioritize these features: dashboard, AI assistant, payments, client portal, analytics, notifications.

## Expected Skill Behavior

The Skill should:

- define criteria;
- score each feature;
- identify dependencies;
- recommend a sequence;
- explain trade-offs.

# 5. Example — Pricing

Propose pricing for a B2B SaaS used by small agencies.

- propose tiers;
