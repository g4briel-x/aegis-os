# 1. Product Strategy Prompt

```text
Act as a senior SaaS product manager.

Task:
Create a product strategy for the provided SaaS idea or product.

Process:
1. Clarify target users.
2. Define the main problem.
3. Define the value proposition.
4. Identify business model implications.
5. Define product pillars.
6. Recommend priorities.
7. Perform 4-pass product validation.

Output:
1. Assumptions
2. Target users
3. Problem statement
4. Value proposition
5. Product strategy
6. Priority initiatives
7. Metrics
8. Risks
9. Validation notes
```

# 2. MVP Scope Prompt

```text
Act as a senior SaaS product manager.

Task:
Define the MVP scope for the given product.

Output:
1. Target customer
2. Core problem
3. Must-have features
4. Excluded features
5. User flow
6. Success metrics
7. Launch risks
8. Next iteration
```

# 3. PRD Prompt

```text
Act as a senior SaaS product manager.

Create a Product Requirements Document.

Output:
1. Context
2. Problem
3. Goals
4. Non-goals
5. Users and personas
6. User stories
7. Functional requirements
8. Non-functional requirements
9. Acceptance criteria
10. Metrics
11. Risks
12. Open questions
```

---

# 4. Prioritization Prompt

```text
Act as a senior SaaS product manager.

Prioritize the provided feature list.

Use explicit criteria:
- user impact;
- business impact;
- effort;
- confidence;
- dependencies;
- risk reduction.

Output:
1. Prioritization method
2. Scoring table
3. Recommended order
4. Rationale
5. Risks and trade-offs
```

# 6. Pricing Prompt

```text
Act as a senior SaaS product manager.

Propose a pricing and packaging model.

Consider:
- customer segments;
- value drivers;
- willingness to pay;
- feature packaging;
- expansion paths;
- competitor alternatives.

Output:
1. Pricing logic
2. Proposed tiers
3. Feature packaging
4. Rationale
5. Risks
6. Validation plan
```
