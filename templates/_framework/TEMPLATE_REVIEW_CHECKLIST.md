# Template Review Checklist

Version: 0.1.0  
Status: Foundation Draft

---

# 1. Purpose

This checklist helps reviewers evaluate a Template before approval.

---

# 2. Review Checklist

```text
[ ] Template id is unique
[ ] Template name is clear
[ ] Output type is explicit
[ ] Purpose is useful
[ ] Inputs are complete
[ ] Variables are documented
[ ] Required sections are clear
[ ] Optional sections are clear
[ ] Usage instructions are practical
[ ] Example is realistic
[ ] Checklist is useful
[ ] Related assets are linked
[ ] Metadata is valid
```

---

# 3. Reviewer Decision

Possible decisions:

```text
approve
request_changes
reject
merge_as_draft
deprecate_duplicate
```

---

# 4. Final Principle

> Template review protects the system from low-quality reusable artifacts.
