# 1. Purpose

The Architecture Decision Record Template provides a reusable structure for documenting important technical and architectural decisions.

It helps product, engineering, architecture, security and operations teams preserve decision context, alternatives, trade-offs, consequences and follow-up actions.

---

# 2. When to Use

Use this Template when:

```text
an architectural decision affects system structure
a technology choice must be recorded
a security-sensitive design choice is made
a database, API or infrastructure decision has long-term impact
a trade-off must be made explicit
future maintainers need decision context
teams disagree and need a documented decision
```

---

# 3. Output

This Template produces:

```text
Architecture Decision Record
```
