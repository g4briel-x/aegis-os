# Go-To-Market Plan Template — Checklists

Version: 0.1.0  
Status: Draft

---

# 1. Completeness Checklist

```text
[ ] Product or offer is defined
[ ] Target segment is defined
[ ] ICP is defined
[ ] Problem is specific
[ ] Value proposition is clear
[ ] Positioning is written
[ ] Offer is defined
[ ] Channels are selected
[ ] Sales motion is defined
[ ] Success metrics are defined
```

---

# 2. Market Quality Checklist

```text
[ ] Segment is narrow enough
[ ] Buyer and user are identified
[ ] Pain is urgent enough
[ ] Current alternatives are known
[ ] Buying trigger is defined
[ ] Differentiators are credible
```

---

# 3. Messaging Checklist

```text
[ ] Primary message is clear
[ ] Supporting messages are specific
[ ] Objections are listed
[ ] Objection responses are credible
[ ] Call to action is clear
[ ] Message matches target segment
```

---

# 4. Execution Checklist

```text
[ ] Launch date is defined
[ ] Launch owner is assigned
[ ] Launch assets are listed
[ ] Channel owners are assigned
[ ] Funnel metrics are defined
[ ] Risks are documented
[ ] Tasks have owners and due dates
```

---

# 5. Final Principle

> GTM checklists should confirm that market focus and execution ownership exist before launch.
