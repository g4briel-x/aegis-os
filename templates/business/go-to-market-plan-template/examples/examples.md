# 1. Example — Audiovisual Financing SaaS

## Product / Offer

```text
Project Readiness Review SaaS
```

## Target Segment

```text
Independent film, documentary and series producers preparing project packages for financing.
```

## Problem Statement

```text
Independent producers often approach financiers without knowing whether their project materials are complete, credible and investor-ready.
```

## Value Proposition

```text
A readiness platform that helps producers identify missing materials and prepare stronger financing conversations before contacting investors.
```

## Offer

```text
Paid pilot for 10 creators with guided project readiness review.
```

## Primary Channels

```text
Founder-led outreach to producer communities
Partnerships with film incubators
LinkedIn content targeting independent producers
```

## Primary Launch Metric

```text
10 qualified creators complete one project readiness review during the pilot.
```

# 2. Example — Sales Motion

```text
founder_led_sales
```

Sales process:

```text
1. Identify producer leads.
2. Send personalized outreach.
3. Run 20-minute discovery/demo call.
4. Invite qualified creators to paid pilot.
5. Convert successful pilot users to subscription.
```

---

# 3. Example — Objection Handling

```text
Objection: I already know what financiers need.
Response: The review does not replace experience; it reduces blind spots by checking whether the package is complete and finance-ready before outreach.
```
