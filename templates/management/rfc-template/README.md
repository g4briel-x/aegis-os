---

# 1. Purpose

The RFC Template provides a reusable structure for proposing, discussing, reviewing and approving significant changes in a SaaS project or in Aegis OS.

It helps product, engineering, architecture, security, operations and leadership teams evaluate proposals before execution.

---

# 2. When to Use

Use this Template when:

```text
a major product change is proposed
a technical direction needs review
a system capability must be added
a new process or governance rule is needed
a breaking change is planned
a cross-team decision requires alignment
a proposal needs comments before approval
```

---

# 3. Output

This Template produces:

```text
Request for Comments Document
```
