# 1. Purpose

The Runbook Template provides a reusable structure for documenting operational procedures that teams must execute consistently during production support, incident response, deployment, maintenance, recovery or recurring operations.

It helps engineering, DevOps, security and support teams reduce ambiguity during high-pressure situations.

---

# 2. When to Use

Use this Template when:

```text
a production operation must be repeatable
incident response requires clear steps
deployment or rollback needs an execution guide
a support team needs safe operational instructions
a maintenance task must be performed regularly
a recovery procedure must be documented
a critical workflow needs an escalation path
```
